# Laravel Inputables
Utility form tags component collection for Blade & Laravel 
### Installation

```bash
composer require aboleon/laravel-inputables`

php artisan vendor:publish --tag=mfw-inputables
```
### Usage
    
```blade
<x-mfw::input name="phone" :value="$data->phone" />
```
